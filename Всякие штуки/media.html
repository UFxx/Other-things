Что такое медиа-запросы

Медиа-запросы применяются для адаптации сайта под экраны устройств. Для каждого устройства, будь то монитор, телевизор или проектор применяется своё "ключевое слово", например: screen, tv, projection и т.д. Но редко кто этим на самом деле пользуется. Можно использовать вместо "ключевого слова" значения ширины экрана. Например, мы хотим сделать что-то для телефона, у которго ширина экрана 480 px, тогда мы пишем:
(max-width: 480px) {
p {
color: red;
}
}
На всех мониторах, у которых ширина 480 px, текст в теге p будет красным. Далее рассмотрим на примерах.

Размеры экранов:
1280px и более (компьютеры),
1024px (планшеты и нетбуки (с альбомной ориентацией)),
768px (планшеты (портретная ориентация)),
600px (небольшие планшеты),
480px (телефоны (альбомная ориентация)),
320px (телефоны (портретная ориентация)).
